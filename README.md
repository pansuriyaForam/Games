# Projects

This folder contains complete Lua projects, including games and simulations.

## Available Projects

### 1. AI-Dungeon-Maze
- **Description**: A puzzle game where AI dynamically adapts to challenge the player.
- **Tech Stack**: Lua, LÖVE2D
- **How to Run**:
  ```bash
  cd AI-Dungeon-Maze
  love .
  ```
- **Features**:
  - AI-driven obstacles and dynamic challenges.
  - Procedural level generation.
  - Immersive puzzle-solving mechanics.

### 2. SnakeGame
- **Description**: A modernized Snake game with a cosmic-themed background.
- **Tech Stack**: Lua, LÖVE2D
- **How to Run**:
  ```bash
  cd SnakeGame
  love .
  ```
- **Features**:
  - Dynamic gradient-colored snake.
  - Starry background with particle effects.
  - High score tracking and sound effects.

### 3. LuaParticlesSim
- **Description**: A simple particle simulation created as a learning project.
- **Tech Stack**: Lua, LÖVE2D
- **How to Run**:
  ```bash
  cd LuaParticlesSim
  love .
  ```
- **Features**:
  - Particles spawn on mouse click.
  - Real-time physics simulation.
  - Basic interactivity with movement and bouncing effects.
