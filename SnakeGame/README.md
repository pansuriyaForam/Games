# Cosmic Snake: A Galactic Adventure 🚀🐍

A modern take on the classic Snake game, built with LÖVE2D. Guide your cosmic snake through a starry universe, collecting food, growing longer, and avoiding collisions. With dynamic visuals, smooth controls, and immersive sound effects, this game brings a fresh experience to a timeless classic.

---

## 🌟 Features

- **Dynamic Gradient Snake** – The snake changes colors dynamically as it grows.
- **Cosmic Background** – A starry space environment with moving particles.
- **Smooth Controls** – Responsive movement for an engaging experience.
- **High Score Tracking** – Keeps track of your best performance.
- **Sound Effects** – Eat food, hit the wall, and experience effects with sound.

---

## 🎮 How to Play

- **Objective**: Eat food to grow longer while avoiding collisions.
- **Controls**:
  - **Arrow Keys** – Move the snake (Up, Down, Left, Right).
  - **R Key** – Restart the game after Game Over.

## 🔧 Installation & Running the Game

### 1️⃣ Install LÖVE2D  
Download and install LÖVE2D from [love2d.org](https://love2d.org/).

### 2️⃣ Clone the Repository  
```bash
git clone https://github.com/pansuriyaForam/Lua.git
```

### 3️⃣ Navigate to the Game Directory  
```bash
cd Lua/LOVE2D/SnakeGame
```

### 4️⃣ Run the Game  
- **Windows**:  
  ```bash
  start love .
  ```
- **Mac/Linux**:  
  ```bash
  love .
  ```
