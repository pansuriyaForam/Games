# LuaParticlesSim 🌌

A simple particle simulation made using LÖVE2D. Click anywhere to spawn colorful particles that move around randomly and fade away over time. The particles bounce off the window edges, creating a cool cosmic effect.

## 🎮 How It Works
- Click anywhere in the window to spawn particles.
- Particles have random sizes, colors, and speeds.
- They fade away over time and get removed when fully transparent.
- Particles bounce off the edges of the screen.

## ⚙️ Controls
- **Left Click** - Spawn particles at mouse position.

## 🛠️ Tech Stack
- **Language:** Lua
- **Framework:** LÖVE2D

## 📌 To Run The Project
1. Install [LÖVE2D](https://love2d.org/).
2. Open a terminal and navigate to the project folder.
3. Run:  
   ```sh
   love .
   ```
