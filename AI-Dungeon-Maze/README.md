# AI Dungeon Maze

## Overview
AI Dungeon Maze is a game developed using **LÖVE2D** and **Lua**, where players navigate through a dynamically generated maze. The goal is to escape while dealing with AI-controlled obstacles and challenges.

## Features
- **Randomly generated maze** for each playthrough.
- **Player movement** using keyboard controls.
- **Basic AI elements** (planned for future updates).
- **Expandable mechanics**, including AI pathfinding and adaptive challenges.

## How to Play
1. Install [LÖVE2D](https://love2d.org/) if you haven't already.
2. Clone or download this repository.
3. Open a terminal in the project folder and run:
   ```sh
   love .
   ```
4. Navigate through the maze using arrow keys or WASD.
